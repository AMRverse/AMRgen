# Table mapping CARD/RGI Model ID and CARD Short Name

Table mapping all CARD/RGI Model ID and CARD Short Name to convert long
names into shortened AMR determinant names

## Usage

``` r
rgi_short_name_table
```

## Format

`rgi_short_name_table` A data frame with 6445 rows and 2 columns:

- `Model ID`: RGI/CARD model ID column

- `CARD Short Name`: Shortened AMR determinant names

## Source

Two columns extracted from: <https://card.mcmaster.ca/latest/data> in
aro_index.tsv
