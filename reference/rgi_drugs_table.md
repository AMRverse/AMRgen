# Table mapping CARD/RGI drug class and antibiotic columns

Table mapping CARD/RGI drug class / antibiotic columns to class /
antibiotic names recognised by AMR pkg

## Usage

``` r
rgi_drugs_table
```

## Format

`rgi_drugs_table` A data frame with 3 rows and 70 columns:

- `RGI_DrugClassAgent`: RGI/CARD drug class / antibiotic that is not
  recognized by AMR pkg

- `drug_class`: Valid drug class name recognised by AMR pkg

- `drug_agent`: Valid antibiotic name recognised by AMR pkg
